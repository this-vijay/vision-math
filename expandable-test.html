# Expandable test
What about masks that don't filter exhaled air? (Click to Expand)
Some N95 masks come with valves or openings that release air without filtering it. The CDC recommends against wearing these masks in public health settings, because they put other people at risk. You can see this for yourself in the interactive above.

First set the inhale & exhale effectiveness to a high value (with mask usage somewhere in the middle). This represents a highly effective mask like an N95. Now drag the exhale effectiveness towards zero and watch how disease transmission rises, particularly among non-mask wearers.

If you happen to already have one of these masks, the CDC recommends covering the exhalation valve with a surgical mask.
